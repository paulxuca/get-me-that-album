# get-me-that-album
Get most albums downloaded directly to your computer!

#Usage
```javascript
npm install get-me-that-album -g
```

#CLI Usage
```javascript
get-me-that-album "ALBUM NAME" "ARTIST" "DESTINATION"
//For Example
get-me-that-album "Views" "Drake" /Users/Me/Downloads
```
